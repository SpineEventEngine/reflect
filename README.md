# Spine Reflect

Spine Reflect is a utility library for working with reflection in Kotlin and Java code.

## Gradle dependency
To use Spine Reflect in your Gradle project:

```kotlin
dependencies {
    implementation("io.spine.tools:spine-reflect:${version}")
}
```
